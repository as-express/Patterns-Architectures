# Laptop CRUD API with Node.js, Express, and MongoDB

This project is a **CRUD (Create, Read, Update, Delete)** API for managing laptops, built using **Node.js**, **Express**, and **MongoDB**. The API supports operations like creating, fetching, updating, and deleting laptop entries, and includes **input validation** using **Joi** to ensure correct data formats.

## Features
- **CRUD Operations**: Create, Read, Update, and Delete laptops.
- **Input Validation**: Validation of incoming data using `Joi` for creating and updating laptops.
- **MongoDB Integration**: MongoDB is used for storing laptop data.
- **SOLID Principles**: The project follows the SOLID principles for clean and maintainable code.

## Technologies Used
- **Node.js** - JavaScript runtime environment.
- **Express.js** - Web framework for building RESTful APIs.
- **MongoDB** - NoSQL database for storing laptop records.
- **Joi** - For data validation and schema enforcement.
